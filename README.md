# Pollution & Fertility in Eastern Europe 
**Team:** Alejandra Parra, Egor Sergeev, Zhanxiong Song
*******
This project was developed during the EDSD course “Using Environmental Data from Remote Sensing in Demographic Analysis” and focuses on combining fertility indicator with air pollution. 
The repository currently contains directories for code, data, and figures, plus an RStudio project file for convenient setup.
